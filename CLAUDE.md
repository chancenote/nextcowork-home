## 하네스: 넥스트코웍 웹사이트 완성도 향상

**목표:** 넥스트코웍 웹사이트를 브랜딩·마케팅 / 세일즈 / 고객경험(CX) 3개 비즈니스 차원 + 교육/컨설팅 상품기획·공간개발 컨설팅 관점으로 진단·개선·기획·구현하여 완성도를 높인다.

**트리거:** 사이트 평가·개선·진단, 웹/홈페이지 개선, 전환율·카피·CTA 개선, 브랜딩 점검, 세일즈·수주 전환·오퍼/가격, 고객경험·CX·사용성·여정·접근성 점검, 3차원 평가(브랜딩·세일즈·고객경험), 상품/오퍼/가격 기획, 페이지 완성도, 사이트 리뉴얼, 시각자산·인포그래픽·구조 도식·SVG 제작 관련 요청 시 `nextcw-site-orchestrator` 스킬을 사용하라. 재실행·업데이트·부분 수정 요청도 이 스킬로 트리거한다. 단순 단발 수정 질문은 직접 응답 가능.
